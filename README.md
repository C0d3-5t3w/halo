# Basic webpage tools
>Tools to create and serve basic webpages.

>Has hot reload on page refresh, no need to stop the server.

### Build and install:

```bash
make all && sudo make install
```

### Uninstall:

```bash
sudo make uninstall
```

### Usage:

```bash
haloinit #creates directories and templates for webpage

halo #starts server with config.json settings
```
